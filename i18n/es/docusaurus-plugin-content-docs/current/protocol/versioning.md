# Versionado

El versionado incremental no es todavía parte del protocolo, pero puede implementarse facilmente en el futuro, ya que sería tan fácil como guardar múltiples índices por cada versión, por ejemplo: rd-index-0.3.json podría compararse con rd-index-0.5.json.

Los SDKs también implementan un método para fusionar índices que se usará en el futuro para este caso.

La compresión también es una característica que no forma parte del protocolo, eso es elección del usuario.
